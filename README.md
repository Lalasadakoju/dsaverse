# DSAVerse 🚀

DSAVerse is a simple Data Structures & Algorithms visualizer built using only HTML, CSS, and JavaScript. It helps users understand core DSA concepts through interactive visualizations.

## 🔥 Features

- Stack Visualization
- Queue Visualization
- Linked List Visualization
- Array Operations
- Tree Visualization
- Graph Basics
- Heap Visualization

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)

## 🎯 Purpose

This project was created to improve understanding of Data Structures through visual learning and hands-on interaction.
